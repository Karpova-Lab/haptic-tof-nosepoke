    Overview#

    This is the documentation for a nosepoke that is used for neuroscience experiments. It was developed in the Karpova Lab at HHMI's Janelia research campus. It is a pyControl compatible device.

    Features#

    • Vibration

      Provide haptic feedback using an integrated vibration motor.

    • Distance

      A time of flight sensor acquires analog distance data that is independent of the object's reflectance.

    • Light

      A controllable white LED illuminates the inside of the nosepoke from above.

    • Liquid delivery

      An integrated channel dispenses liquid to the front of the port. 1/8" tubing can easily be attached using the push-to-connect fitting.
